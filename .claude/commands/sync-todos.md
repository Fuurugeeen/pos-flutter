# sync-todos コマンド

doc/todo-tmp.md の修正指示を doc/todo.md に統合します。

## タスク手順

1. **doc/todo-tmp.md の内容を読み取る**
   - 修正指示を精査し、内容を理解する
   - 各修正項目を明確に識別する

2. **doc/todo.md に反映する**
   - todo-tmp.md の修正指示を todo.md に追加
   - 既存の TODO 項目と重複しないよう確認
   - 優先度や依存関係を考慮して適切な位置に配置
   - 各項目に状態（未着手/進行中/完了）を設定

3. **todo-tmp.md の項目を完了済みにマークする**
   - 反映した項目に ✅ または [完了] マークを追加
   - 反映日時を記録
   - 元の内容は保持（削除しない）

4. **検証と確認**
   - todo.md に全ての項目が正しく反映されているか確認
   - todo-tmp.md の全ての項目が完了マークされているか確認
   - 両ファイルの整合性を確保

5. **結果レポート**
   - 反映した項目数を報告
   - 新規追加、更新、スキップした項目の詳細を表示
   - 次のアクションを提案

## 注意事項

- 修正指示が不明確な場合は、ユーザーに確認を求める
- todo.md の既存項目は上書きせず、更新または追加として扱う
- 完了済みの項目は todo.md では別セクションに移動する
